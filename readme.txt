== Description ==

Adds a customizeable widget which displays the latest HSDPA UMTS Prepaid News by [http://www.hsdpa-umts-prepaid.de/](http://www.hsdpa-umts-prepaid.de/)

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==

1. Extract `hsdpa-umts-prepaid-news.zip`.
2. Upload the `hsdpa-umts-prepaid-news`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.

*German*
a. Entpacke `hsdpa-umts-prepaid-news.zip`.
b. Lade den `hsdpa-umts-prepaid-news`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
c. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
d. F&uuml;ge das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
